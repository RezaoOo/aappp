# 🔒 VPN Simple

یک اپ VPN ساده و سریع برای Android با V2Ray

## 🎯 ویژگی‌ها
- ✅ یک دکمه قطع/وصل ساده
- ✅ سیستم خودکار پینگ و انتخاب بهترین سرور
- ✅ UI شیک Apple-style
- ✅ پشتیبانی از V2Ray URIs
- ✅ بنر تبلیغاتی (در حال آپدیت)

## 📦 نیازمندی‌ها
- Android 21+
- Android Studio Flamingo+

## 🚀 شروع سریع

```bash
git clone https://github.com/RezaoOo/aappp.git
cd aappp
./gradlew build
```

## 📝 معماری
- **UI**: Jetpack Compose (Material 3)
- **Config**: JSON-based
- **Coroutines**: Async operations
